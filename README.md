# Library-Admin-Panel

A small website featuring the following:
-An admin panel for interacting with an SQL Database.
-Login and authentication system with password encryption.
-No use registration, users are added by the site admin.

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Deploy

```bash
-gcloud init
-gcloud app deploy
```
